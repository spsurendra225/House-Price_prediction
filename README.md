🏠 House Price Prediction

📌 Overview

This is a beginner Machine Learning project that predicts house prices using Linear Regression.

⚙️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

📊 Features

- Area
- Bedrooms
- Bathrooms

🚀 How to Run

1. Install requirements:
   pip install -r requirements.txt

2. Run the code:
   python house-price-prediction.py

📈 Output

- Shows predicted vs actual graph
- Displays R² score

🎯 Purpose

To learn basic Machine Learning workflow and model building.

🔮 Future Improvements

- Use larger dataset
- Add multiple models
- Build web app using Flask
